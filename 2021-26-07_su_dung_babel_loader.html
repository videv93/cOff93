Cách sử dụng babel loader

Tải thử viện babel-plugin-module-resolver từ npmjs.com

Tạo file config .babelrc. Sau đó copy/paste đoạn code sau 

{
            "plugins": [
              ["module-resolver", {
                "root": ["./src"],
                "alias": {
                  "test": "./test",
                }
              }]
            ]
          }
